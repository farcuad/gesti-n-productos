# Sistema de Gestión de Productos (Frontend)

Este es el cliente web para el Sistema de Gestión de Productos Multi-tenant. Una interfaz moderna, rápida y segura diseñada para que cada empresa o cliente gestione su propio inventario de forma aislada.

---

##Tecnologías Utilizadas

* **Framework:** [React.js](https://reactjs.org/).
* **Estilos:** [Bootstrap 5](https://getbootstrap.com/) para un diseño responsivo y profesional.

---

## Características Principales

- **Arquitectura Multi-tenant:** Acceso restringido y datos aislados por cliente.
- **Gestión CRUD:** Creación, lectura, actualización y eliminación de productos.
- **Con ventas y agregar trabajadores** Hace ventas, agregar trabajadores
- **Interfaz Responsiva:** Optimizado para dispositivos móviles y escritorio.
- **Dashboard de Control:** Visualización clara del stock y estados de productos.

---

## Configuración del Entorno

Para ejecutar este proyecto localmente, asegúrate de tener instalado [Node.js](https://nodejs.org/).
